2.1. Primer inicio de Windows

Para iniciar Windows, podemos hacerlo directamente del menú de inicio que hemos modificado antes, o ejecutando el programa WIN.EXE en la carpeta C:\WINDOWS. Ya que tenemos los drivers del protocolo en una ISO, arrancamos con los drivers de CD y ejecutamos Windows.

La primera vez que arrancamos Windows, con los drivers de red, nos pedirá que creemos un archivo de contraseña. No es obligatorio, pero es recomendable para iniciar sesión con una contraseña. La creamos e indicamos la nueva contraseña.






2.2 Instalar Protocolo

Para instalar el protocolo TCP/IP, vamos a la carpeta Principal, Instalar Windows, Opciones, Cambiar la configuración de red...



Abrimos la lista de controladores y vamos a Agregar protocolo...



Seleccionamos Protocolo no listado o actulizado e indicamos donde está el driver.





2.3. Configurar

Una vez instalado, configuramos el protocolo TCP/IP para indicar la IP y la puerta de enlace de la red (en este caso la IP sería 192.168.5.41 y la puerta de enlace 192.168.5.1), y establecer el protocolo TCP/IP como predeterminado.



Una vez configurado todo, nos pedirá reiniciar el sistema. Después la red funcionará correctamente.
