4. Demostración de compartición de archivos


Una de las características de Windows 3.11 es la de compartir archivos por red. Un ordenador comparte una carpeta, y el otro la monta como una unidad de red y la puede usar en el sistema como otra unidad (con los permisos establecidos por el que la comparte).

Para compartir archivos en Windows 3.11, en la máquina donde están los archivos vamos a la carpeta deseada, y en Disco seleccionamos Compartir directorio. Podemos modificar las opciones de acceso a nuestro gusto. Después en la otra máquina, vamos a Disco, Conectar a unidad de red, y seleccionamos el PC en la red y la carpeta que comparte. Si no aparece en la lista, se puede escribir manualmente.
