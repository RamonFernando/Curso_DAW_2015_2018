1.1. Preparar VirtualBox

Para poder instalar el sistema operativo Windows 3.11, debemos tener de base una versión del sistema MS-DOS. En este ejemplo se usará una máquina virtual con MS-DOS 6.22 instalado con algunos añadidos (CD, tarjeta de sonido, arranque con selección, etc.).

Si queremos usar la red en Windows 3.11, primero deberemos configurar Virtualbox para que la máquina virtual tenga una tarjeta de red compatible, y crear una red a donde conectar las máquinas.

En la configuración de Virtualbox, en el apartado de red, creamos una nueva Red NAT, la configuramos y ponemos una red de IP (por ejemplo) 192.168.5.0 con máscara de 24 bits (255.255.255.0).



Después, en la máquina virtual, hay que instalar una tarjeta de red compatible. Vamos a la configuración de la máquina, y en el apartado de Red abrimos las opciones avanzadas. En Tipo de adaptador elegimos la tarjeta PCNet-PCI II (Am79C970A).



Una vez configurado, podemos proceder a la instalación de Windows 3.11.



1.2. Instalación del sistema

Arrancamos la máquina virtual, e iniciamos con los drivers de CD (ya que los drivers de la tarjeta de red los tenemos en un .ISO).



Una vez el sistema MS-DOS esté iniciado, insertamos el disquete 1 de 9 de la instalación de Windows 3.11, nos vamos a la unidad A:, y ejecutamos el programa INSTALAR.EXE.

Seguimos las instrucciones que salen en pantalla, y seleccionamos la instalación rápida (básicamente, darle dos veces a Enter). En la instalación nos pedirá que cambiemos el disquete. Cambiamos este por el pedido cuando sea necesario.




Después, aparecerá la interfaz gráfica del instalador de Windows. Rellenamos la información que nos pide y seguimos con la instalación como antes.



Como no tenemos impresora conectada, instalamos la opción No hay impresoras conectadas.





1.3. Instalar tarjeta de red

En la instalación de Windows se puede instalar directamente la tarjeta de red antes del primer inicio.

Ya que la tarjeta de red de la que disponemos es más moderna que el sistema, debemos instalar drivers manualmente. Seleccionamos de la lista Adaptador de red no listado o actualizado, insertamos el CD de drivers (la imagen ISO en el VirtualBox), y seleccionamos la carpeta donde se encuentran los drivers.



Por último, falta indicar el nombre del usuario, el nombre del PC en la red, y terminar la instalación de Windows con los dos disquetes restantes.



Al finalizar, nos avisará de que los archivos CONFIG.SYS y AUTOEXEC.BAT están modificados y se han creado copias con las líneas necesarias para ejecutar la red de Windows. Estas copias están en C:\WINDOWS y se llaman CONFIG.WIN y AUTOEXEC.WIN. Hay que sustituir estos con los archivos originales, y modificarlos para que haya una nueva opción que ejecute automáticamente Windows 3.11 con las líneas incluidas de los drivers de red.
