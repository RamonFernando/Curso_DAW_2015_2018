3. Ping entre máquinas


Con un clon de la máquina virtual, y en este clon cambiadas las direcciones MAC e IP, las máquinas se podrán comunicar entre sí. Para probarlo, arrancamos Windows y vamos a la terminal MS-DOS. El comando es ping IP. Para ver la IP de la máquina podemos usar el comando ipconfig.
